# PR-title

> A GitHub App built with [Probot](https://github.com/probot/probot) that Check PR title format

## Setup

https://github.com/apps/pr-title

To configure add a `.github/pr-title.yml`


```pr-title.yml
regex: ^PR-TITLE.*$%
message: Custom message

```
